# 🔤 ABC Flashcards App

Aplicación interactiva de tarjetas educativas para el aprendizaje del abecedario, combinando lógica de estados y una interfaz moderna.

## ✨ Características
- **Interacción 3D:** Efecto de volteo (*flip*) dinámico mediante CSS.
- **Contador Dinámico:** Seguimiento en tiempo real de cartas descubiertas.
- **Feedback Auditivo:** Integración de sonidos para mejorar la experiencia de usuario.
- **Visuales Optimizados:** Imágenes adaptativas con `object-fit` y alineación simétrica.
- **Responsive Design:** Grid flexible compatible con móviles y escritorio.

## 🛠️ Tecnologías
- **HTML5 / CSS3** (Custom Grid & Animations)
- **Bootstrap 5.3**
- **JavaScript Vanilla** (Manipulación del DOM y Eventos)
